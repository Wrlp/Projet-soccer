# Frontend SportInsight

```bash
# Depuis la racine du projet (recommandé)
./run-dev.sh

# Ou seulement le front
npm install && npm run dev
```

API sur http://localhost:8000 — proxy Vite `/api`.

Pages : upload, timeline, événements, matrice (projet), entraînement (projet).

`VITE_USE_MOCK=true` dans `.env` pour mode démo sans API.
